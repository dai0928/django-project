HTMLのバージョンについて

これから作るサイトはHTML5で大丈夫だが、昔に作られたサイトや、古いブラウザでは、HTMLのバージョンが低い場合もある

以下は、主に使われるHTMLのバージョン

HTML4.01 XHTML1.0 HTML5

DOCTYPE宣言について

基本的にバージョンが変わっても使える属性、要素などが変わるだけなので、今どのバージョンのHTMLでコードを書いているか宣言する必要がある

HTMLはバージョン毎に使用可能な属性、要素などのルールが記述されたファイルがある

そしてそのファイルのことを、　Document Type Definition で「DTD」と言う。

DTDのファイルは、HTMLの創設サイトのW3C内にある。書きたいバージョンのDBDをサイトからコピーして、htmlコードの最初に記述する

以上がDOCTYPE宣言になる　DOCTYPEは、document type のこと。

ただし、HTML5にはDTDファイルが存在しない

HTML5にはDTDが存在しない為、DOCTYPE宣言は必要ないのだが、ブラウザによってレイアウトが崩れることがある

そのため、コードの冒頭に以下の最低限のDOCTYPE宣言をしておくとよい

!DOCTYPE html

DOCTYPE宣言には大文字小文字を区別しないので、どちらで記述してもよい

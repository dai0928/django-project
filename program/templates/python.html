Python導入

Pythonは、プログラミング言語の中でも非常に人気のある言語です。
AIや機械学習、ネットにある情報を選択して抽出するWEBスクレイピングなど様々な事ができます。
また、「Django」や「Flask」といったフレームワークでのWEBサービスの開発もでき、
このページはその「Django」を使用して作成しています。

開発環境の構築

まずは、PC上でPythonで開発するための環境を構築しましょう。
以下のリンクからPythonの公式サイトにアクセスしてください。

https://www.python.org

アクセスしたら、「Downlords」のタブをクリックすると、
Downlord Pthon 〇〇のボタンがあるのでそこからダウンロードができます。

ダウンロードしたファイルを開くと以下のようなセットアップの画面が出て来ます。

画面下部に、「Add Python 3.6 to PATH」というチェックボックスがありますが、
これを必ずチェックしてからインストールしましょう。

テキストエディタの実装

Pythonのインストールが終わったら、次はコードを書くためのテキストエディタを入れましょう。
「Visual Studio Code(VS-CODE)」、「PyCharm」、「Atom」、「Sublime text」など様々なエディタがありますが、
料金や拡張機能などを調べて自分に合ったものを使うとよいでしょう。
私は、HTMLのコーディングには「VS-CODE」、Pythonでは「PyCharm」を使用しています。

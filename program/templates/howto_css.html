cssについて

前回まではHTMLの講座をしていましたが、この「css」を使うことによって、
HTMLで書いた文字のフォントを変えてみたり、ページの背景に枠をつけてみたり、ボタンの上に カーソルを持っていくとそのボタンに動きをつけたりなどいろいろなことが出来るようになります。

cssの指定方法

cssの指定には「別にcss専用の外部ファイルを作り、そのファイルをHTMLファイルで指定する方法と 「HTMLに直接記述する方法」の２種類存在します。

外部ファイルを指定する方法

HTMLのファイルは、拡張子が「.html」でしたが、cssのみを記述したファイルの拡張子は「.css」となります。
新たにcssファイルを作成して、HTMLファイルで、<link>タグでそのcssファイルを指定することができます。
実際に記述すると、以下のようになります。

HTML

                
    
                    
  1.    <head>
    2. 
                    
  2.      <mata charaset="UTF-8">
    3. 
                    
  3.        <link rel="stylesheet" href="static/main.css" type=text/css>
    4. 
                    
  4.    </head>
    5.

HTMLの<a>要素でもやったように、「href」属性を使って、cssファイルまでのパスを指定しています。

cssの書き方

style属性でHTMLの要素に記述する

まず、<p>要素や<h1>要素や<body>要素などに、style属性で指定する方法です。
;（セミコロン）で区切ることによって、複数のcssを書くことが可能です。

HTML

                    
    
                        
  1.    <body>
    2. 
                        
  2.      <h1 style="color: red; font-size: 30px;">
    3. 
                        
  3.      この文章にはCSSを当てています。
    4. 
                        
  4.      </h1>
    5. 
                        
  5.    </body>
    6.

結果

                    
この文章にはCSSを当てています。

<head>要素内に<style>要素で記述する

この方法で記述すると、複数の要素を指定して一括でcssを当てたりすることができます。

HTML

                    
    
                        
  1.    <head>
    2. 
                        
  2.      <style type="text/css">
    3. 
                        
  3.      h1 {
    4. 
                        
  4.          color: red;
    5. 
                        
  5.          font-size: 30px;
    6. 
                        
  6.         }
    7. 
                        
  7.      </style>
    8. 
                        
  8.    </head>
    9. 
                        
  9.    <body>
    10. 
                        
  10.      <h1>この文章にはCSSを当てています。</h1>
    11. 
                        
  11.      <h1>すべてのh1要素にcssを指定しています。</h1>
    12. 
                        
  12.    </body>
    13.

結果

                    
この文章にはCSSを当てています。

                    
すべてのh1要素にcssを指定しています。

別のcssファイルで記述し、指定する

この方法は、様々なHTMLファイルと共有して使えたり、 cssを修正したいときなどに一括で修正できたりするので使い勝手がいいです。

HTML

                    
    
                        
  1.    <head>
    2. 
                        
  2.      <mata charaset="UTF-8">
    3. 
                        
  3.        <link rel="stylesheet" href="static/main.css" type=text/css>
    4. 
                        
  4.    </head>
    5. 
                        
  5.    <body>
    6. 
                        
  6.      <h1>別のファイルでcssを指定しています。</h1>
    7. 
                        
  7.    </body>
    8.

main.css

                    
    
                        
  1.    h1 {
    2. 
                        
  2.        color: red;
    3. 
                        
  3.        font-size: 30px;
    4. 
                        
  4.       }
    5.

結果

                    
別のファイルでcssを指定しています。

セレクタ

上記の書き方でやった１番目以外の方法では、セレクタというもので要素を指定してcssを適用しています。 上記のcssを例に挙げると、「h1 {}」の「h1」の部分がセレクタになります。他にも様々な指定方法があります。

  • 要素名を直接指定
    h1 {}, p {}, a {}など
  • 要素名を直接指定+複数指定
    h1, p {}
  • ul要素の中のli要素を指定
    ul li {}
  • id="check"の属性を持つ要素を指定
    #check {}
  • class="check"の属性を持つ要素を指定
    .check {}

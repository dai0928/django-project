Lesson6~箇条書きについて~

箇条書きには、項目の順番が決まっていない「順不同型」と順番が重要な「番号順」の箇条書きが存在します。

<li>要素

<li>要素は、list item の略でリストの項目を表す要素です。
実際に記述すると、以下のようになります。

HTML

                            
    
                                
  1.    <body>
    2. 
                                
  2.      <p>例&nbsp;卵かけご飯の材料</p>
    3. 
                                
  3.        <li>卵</li>
    4. 
                                
  4.        <li>ごはん</li>
    5. 
                                
  5.        <li>醤油</li>
    6. 
                                
  6.    </body>
    7.

結果

                            
例 卵かけごはんの材料

                            
  • 卵
    • 
                            
  • ごはん
    • 
                            
  • 醤油
    •

    <li>のみでは、リストが順不同型なのか番号順の物なのかわからないので、それを示す要素が必要になります。

    <ul>要素

    <ul>要素は、unorderd list の略で、順不同のリストを示す要素で、 カテゴリがフローコンテンツ、コンテンツモデルは０個以上のli要素です。
    実際に記述すると、以下のようになります。

    HTML

                                
      
                                
    1.    <body>
      2. 
                                
    2.      <ul>
      3. 
                                
    3.        <li>卵</li>
      4. 
                                
    4.        <li>ごはん</li>
      5. 
                                
    5.        <li>醤油</li>
      6. 
                                
    6.      </ul>
      7. 
                                
    7.    </body>
      8. 
                                
    8. </html>
      9.

    結果

                                
      
                            
    • 卵
      • 
                            
    • ごはん
      • 
                            
    • 醤油
      •

    上記のように、ul要素を入れると、各リストの行頭に「・(行頭記号)」が自動的に入ります。

    <ol>要素

    これはul要素とは逆で、orderd list の略で番号順のリストを示す要素で、カテゴリなどはulと同じです。
    HTMLの書き方も上記のulと同じです。

    HTML

                                
      
                                
    1.    <body>
      2. 
                                
    2.      <p>卵かけごはんの作り方</p>
      3. 
                                
    3.      <ol>
      4. 
                                
    4.        <li>ご飯をよそう</li>
      5. 
                                
    5.        <li>ごはんの上に卵を割る</li>
      6. 
                                
    6.        <li>醤油をかけて混ぜる</li>
      7. 
                                
    7.      </ol>
      8. 
                                
    8.    </body>
      9. 
                                
    9. </html>
      10.

    結果

                                
    例　卵かけごはんの作り方
    
                            
      
                                
    1. ごはんをよそう
      2. 
                                
    2. ごはんの上に卵を割る
      3. 
                                
    3. 醤油をかけて混ぜる
      4.

    ol項目を入れると、ページ上でリストの上から順番に数字が自動入力されます。

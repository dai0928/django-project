Pythonの基礎文法

まずは、前回ダウンロードしたテキストエディタを開きます。
そうしたら、好きなフォルダに「test.py」というファイルを作りましょう。
※Pythonの拡張子は.pyになります。
そしたらそのテキストエディタで、test.pyを開きましょう。

Hello World!

Pythonで、hello world!を実行してみましょう。
エディタで開いている.pyのファイルに次のように記述します。

test.py

            
    
              
  1. print("hello world!")
    2.

そしたら、そのファイルを実行してみましょう。
まず、Windowsを使用しているなら「Windows Power Shell」、
Macの場合は「ターミナル」を開いて、

              
    
                
  1. cd 〇〇
    2.

〇〇にtest.pyのファイルがあるフォルダのパスを入れて実行し、
そこで、 

              
    
                
  1. python test.py
    2.

で実行します。そうしたら、

              
    
                
  1. python test.py
    2. 
                
  2. hello world!
    3.

と帰ってきます。
「print()」とは、pythonの関数で、()内に入力された情報をコマンドラインに返します。
今回のようにただの文字列を入力する際は、「" "」か「' '」の中に記述する必要があります。

変数

test.py

                
    
                    
  1. a = "emokiti"
    2. 
                    
  2. print(a)
    3.

上記のように、書くと、「a」は「emokiti」という文字列を持った変数となります。
これを実行すると、

                
    
                    
  1. python test.py
    2. 
                    
  2. emokiti
    3.

と帰ってきます。
この「変数」は、使いまわしたい文字列や値を格納する事で、簡単に引き出すことができます。

if文

test.py

                
    
                    
  1. a_length = 160
    2. 
                    
  2. if a_length < 150:
    3. 
                    
  3.     print("small boy")
    4. 
                    
  4. elif a_length > 190:
    5. 
                    
  5.     print("big boy!")
    6. 
                    
  6. else:
    7. 
                    
  7.     print("nomal boy")
    8.

このファイルを実行すると、

                
    
                    
  1. python test.py
    2. 
                    
  2. noaml boy
    3.

と出力されます。説明すると、 「もしa_lengthの値が150より小さかったらsmall boyと出力」
「もし190よりも大きかったらbig boy!と出力」
「そうでなければ（150~190ならば）nomal boyと出力」
といった意味になります。if文とは、このように、
「もし〇ならば▲（if）、★ならば■（elif）、どちらでもなければ◇（else）」
といったように条件をつけて結果を分岐させることができます。

リスト、for文

test.py

                
    
                    
  1. program_list = ["python", "javascript", "PHP", "C"]
    2.

  1. python test.py
    2.

  3. python
    4. 
                    
  4. javascript
    5. 
                    
  5. PHP
    6. 
                    
  6. C

1行目のように、[]の中に「,（カンマ）」で区切った複数の値を変数として持たせることができ、
このようなものを「リスト」といいます。
リストは、左から、０，１，２，３という順番となっており、変数を呼び出す際に後ろに[]で数字を指定する事で、
その順番の値を呼び出すことができます。
3行目のように、for 〇〇(好きな文字列) in □□(リスト)と書くことで、設定した文字列を呼び出すと、０から順番にリストの値を1つずつ取り出すことが可能です。

test.py

                
    
                    
  1. class Card:
    2.

    3. 
                    
  3.         self.name = name
    4. 
                    
  4.         self.age = age
    5. 
                    
  5.         self.gender = gender
    6. 
                    
  6.                         
    7.

  9.

この「クラス」というものは、よく「設計書」に例えられます。
上記の例だと、クラス「Card」の中には、「名前(name)、年齢(age)、性別(gender)」といった1人の人物を構成
するための情報を格納することができます。つまり「Card」は1人の人物を設計するための設計書といえます。
このクラスを元に作成した10行目の「emo」を「インスタンス」と呼びます。
2行目の関数は「コンストラクタ」といい、インスタンスを生成する際に呼び出される関数です。
コンストラクタの第１引数には、必ず「self」を指定して、以降に設定したい情報を引数として指定します。
コンストラクタの働きは、例えば１０行目のようにクラスが呼び出されると、コンストラクタの引数として指定されていた
「name, age, gender」が初期化され、代わりに「"エモ吉", "25", "男"」が引数に代入されます。
第１引数で指定されていた「self」は「自身」という意味で、「emo.name」だと「emoのインスタンス自身が持っているname部分」
ということになります。

import文

importを使うと、外部から既に機能をまとめられた関数を使えるようになります。
例えば、WEBスクレイピングのプログラムを組む場合は、「Selenium」や「BeautifulSoup」などといった
WEBスクレイピングをする為の機能が既にまとめられた「ライブラリ」を使うことによって、簡単にWEBスクレイピング
のプログラムを組むことができます。

test.py

                
    
                    
  1. import chromedriver_binary
    2. 
                    
  2. from selenium import webdriver
    3. 
                    
  3. import random as rd
    4. 
                    
  4. from .test import *
    5.

2行目のfrom~は、「selenium」というライブラリから「webdriver」のみをインポートする
という意味で、 3行目のas~は、「random」というモジュールを「rd」としてインポートするという意味になります。
4行目は値に*（アスタリスク）を指定することで、.testの中の全ての関数をインポートするという意味になります。

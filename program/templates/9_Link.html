Lesson9 ~ハイパーリンク~

HTMLには他のページに飛ぶことができる「ハイパーリンク」という要素があります。

<a>要素

<a>要素はanchor(アンカー)の略で、ハイパーリンクを表す要素です。
カテゴリはフローコンテンツ、フレージングコンテンツ、インタラクティブコンテンツで、
コンテンツモデルはトランスペアレントというものになります。 トランスペアレントとは「透過」という意味で、親要素のコンテンツモデルを引き継ぐという特性を持っています。
例えばp要素の子要素としてa要素を記述した場合、a要素のコンテンツモデルは、p要素と同じ「フレージングコンテンツ」ということになります。

<a>要素はハイパーリンクにしたい範囲の文字列を<a>要素で囲い、 「href」という属性をつけて、属性値としてリンク先（hrmlファイルのパスなど）を指定します。
※hrefはハイパーテキストリファレンス略です。
属性値は、リンクさせたいhtmlファイルが同じディレクトリ（フォルダ内）にあれば、ファイル名のみの記述で大丈夫です。
ハイパーリンクとしたテキストは、「アンカーテキスト」と呼ばれます。
すでに存在するWEBサイトにリンクさせたい場合は、属性値にそのサイトのURLを記述すればリンクさせることができます。
画像にもハイパーリンクをつける事ができるのですが、ハイパーリンク化された画像のことを「バナー」と呼びます。

リンク先を別ウィンドウで表示させる

リンク先を新しい別のウィンドウで表示させたい場合、<a>に属性と属性値としてtarget="_brank"を記述する と、ハイパーリンクを開いた際に、新しいウィンドウで表示されるようになります。

ページ内リンク

例えばブログなどの1つのページが長いページ内で、ほかの文章の所に一気に飛びたい場合に、ページ内リンクが使われます。
このページ内リンクでは、飛びたい文章に「id」という固有の名前を付けることで、href属性でその文章の「id」を指定することができるようになります。
idを指定する際は、頭に「#」をつけてから名前を指定します。

使用例

HTML

                        
    
                            
  1.    <body>
    2. 
                            
  2.        <a href="C:/Users/xxx/anchor.html">これがアンカーテキストです</a>
    3. 
                            
  3.        <a href="C:/Users/xxx/anchor.html" target="_brank">新しいウィンドウで表示されます</a>
    4. 
                            
  4.        <p id="text">この文章にtextという名前をつけています</p>
    5. 
                            
  5.        <a href="#text">このように「#」でidを指定します</a>
    6. 
                            
  6.        <a href="C:/Users/xxx/anchor.html">
    7. 
                            
  7.        <img src="../HTML画像表示用/犬.jpeg" alt="バナー" width="140" height="140">
